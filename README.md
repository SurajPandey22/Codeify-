1️⃣ Clone Repository
git clone https://github.com/your-username/codeify.gitcd codeify

🔧 Backend Setup
cd backendnpm install
Create .env file inside backend:
PORT=5000MONGO_URI=your_mongodb_connectionJWT_SECRET=your_secret_keyREDIS_URL=your_redis_urlCLOUDINARY_CLOUD_NAME=your_cloud_nameCLOUDINARY_API_KEY=your_api_keyCLOUDINARY_API_SECRET=your_api_secretGEMINI_API_KEY=your_ai_api_key
